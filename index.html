The agent runs a series of intelligent algorithms that distinguish the different shapes in each image, constructing knowledge representations of their characteristics and relationships to other shapes and figures. IoT: SHM

Capstone group project for senior design course. SHM is a Structural Health Monitoring system with an internet of things approach. Arduino nodes, placed throughout a structure, communicate with a central Raspberry Pi which sends payloads of accelerometer data to a server running machine learning algorithms. Data is then stored in a MongoDB database and displayed in an intuitive Django dashboard, showing the user the real-time structural health of the building.

Technologies used: Python, C++, MongoDB, Storm, Kafka, Raspberry-Pi, Arduino
